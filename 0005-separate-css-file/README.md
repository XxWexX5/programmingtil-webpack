STARTING OFF:
Make sure you have npm installed.

Install dependencies:
```
$ npm install
```

Run:
```
$ npm run clean
$ npm run compile
$ npm start
```

View at: http://localhost:8080/

CONCEPTS:
webpack separate stylesheets
